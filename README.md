# Mobile-app

I have done this application for a school project, it is an application that allow you to learn different specific characteristics

This application is little complex but no imposible if you had knowledge about react

#Main technologies used

- [React Native](https://github.com/facebook/react-native)

> A framework for building native apps with React.

#Starting the project

To start we create the project on a local or if you want you can clone this project

- Clone this project
```
git clone < project-url.git >
```
-Install NodeJs

#Basic Commands to the project

- Install all the dependencies
```
npm install
```
-Instal an specific dependence
```
npm install <name of the dpendence>
```
-Run the aplication
```
npm run android
```
